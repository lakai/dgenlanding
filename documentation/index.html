Installation

Running a local server

In order to install you need to have Node version less than v15 otherwise it will not work. This has been tested up to v14.18.1 Node version.

Installing Node on Windows

For Windows 64 bit please download from here https://nodejs.org/download/release/v14.18.1/node-v14.18.1-x64.msi

For Windows 32 bit please download from here https://nodejs.org/download/release/v14.18.1/node-v14.18.1-x86.msi

Please open the .msi file to start the installation process.

Installing Node on Mac

For Mac OSX please download from here https://nodejs.org/download/release/v14.18.1/node-v14.18.1.pkg

Please open the .pkg file and follow the Apple Installer process.

Installing Node on Linux

I recommend installing NVM to use 14.18.1 of Node. You can find the installation instructions here https://github.com/nvm-sh/nvm

After installing, please run

nvm install 14.18.1
nvm use 14.18.1

Install Gatby CLI

npm install gatsby-cli -g

Run the following commands in the template's directory

npm install
gatsby develop

Running a live server

The fastest way to get a live server running is by building and copying the files from public directory to your web directory. To build, run the following command

gatsby build

More details can be found here https://www.gatsbyjs.org/docs/deploying-and-hosting/

Portfolio

The Portfolio sections are located at /src/sections/portfolio/

The Portfolio content are located on /content/portfolio directory which are fetched using graphql

---
id: 1
image: './image.jpg'
title: "Front-end Development"
category: "React.js"
link: "https://www.github.com"
---

The image needs to be placed in the same directory
